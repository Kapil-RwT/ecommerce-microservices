## Saga Pattern Flow (Order Service)

┌─────────────────────────────────────────────────────────────────────────────┐
│                          ORDER SAGA ORCHESTRATION                           │
└─────────────────────────────────────────────────────────────────────────────┘

    ┌─────────────┐
    │ Create Order│
    │  (PENDING)  │
    └──────┬──────┘
           │
           ▼
    ┌─────────────────────┐
    │ Reserve Inventory   │──────┐
    │ (Pessimistic Lock)  │      │
    └──────┬──────────────┘      │ FAILURE
           │ SUCCESS              │ ↓
           ▼                      │ Compensate

## Data Flow - Create Order

Client → API Gateway → Order Service
                          │
                          ├──→ Product Service (Get Product Details)
                          │    └──→ PostgreSQL
                          │
                          ├──→ Inventory Service (Reserve Stock)
                          │    └──→ PostgreSQL (with Lock)
                          │
                          ├──→ Payment Service (Process Payment)
                          │    ├──→ PostgreSQL
                          │    └──→ Kafka (payment-events)
                          │
                          ├──→ Inventory Service (Confirm)
                          │    └──→ PostgreSQL
                          │
                          └──→ Kafka (order-events)
                               └──→ Notification Service
                                    └──→ Send Email/SMS

## Security Flow

┌─────────────────────────────────────────────────────────────────────────────┐
│                          SECURITY ARCHITECTURE                              │
└─────────────────────────────────────────────────────────────────────────────┘

    Client Login
         │
         ▼
    User Service
         │
         ├──→ Validate Credentials (BCrypt)
         │
         ├──→ Generate JWT Token
         │    (24 hour expiration)
         │
         └──→ Return Token to Client

    Client Request with Token
         │
         ▼
    API Gateway
         │
         ├──→ Extract JWT from Header
         │
         ├──→ Validate Token
         │    (Signature, Expiration)
         │
         ├──→ Extract User Info
         │
         └──→ Forward to Service
              (with X-User-Id header)

📍 OPTION 1: GitHub Codespaces (RECOMMENDED - Easiest & Fastest)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   Time: 10 minutes | Difficulty: ⭐☆☆☆☆ | Cost: FREE

   Step 1: Create GitHub Repository
   ─────────────────────────────────
   Go to: https://github.com/new
   Repository name: ecommerce-microservices
   Visibility: Public (or Private)
   ⚠️  DO NOT initialize with README, .gitignore, or license
   Click: "Create repository"

   Step 2: Push Your Code
   ──────────────────────
   cd /Users/kapil.rawat/Documents/Personal/ecommerce-microservices
   git add .
   git commit -m "feat: E-Commerce Microservices Platform with Spring Boot"
   git remote add origin https://github.com/YOUR_USERNAME/ecommerce-microservices.git
   git branch -M main
   git push -u origin main

   Step 3: Open in Codespaces
   ──────────────────────────
   1. Go to your repository on GitHub
   2. Click the green "Code" button
   3. Click "Codespaces" tab
   4. Click "Create codespace on main"
   5. Wait 2-3 minutes for environment setup

   Step 4: Start Services
   ──────────────────────
   In the Codespaces terminal:
   ./build-all.sh
   docker-compose up -d

   Step 5: Access Services
   ───────────────────────
   - Check the "PORTS" tab in VS Code
   - Click on port 8761 to see Eureka Dashboard
   - Click on port 8080 to access API Gateway
   - Click on port 3000 for Grafana

   🎉 DONE! Your platform is running in the cloud!
